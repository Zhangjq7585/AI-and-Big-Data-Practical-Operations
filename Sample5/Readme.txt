sample5:AI模型安全实战
	目标：对提供的图像分类模型进行白盒对抗攻击，生成对抗样本使其误分类。
	任务：
		1．加载预训练模型并分析其结构；
		2．使用FGSM或PGD算法生成对抗样本；
		3．确保对抗样本在视觉上与原图一致；
		4．提交对抗样本图片、攻击代码。

1.对抗攻击:在输入数据上添加精心设计的微小扰动，使机器学习模型做出错误预测的技术。这些扰动对人眼几乎不可察觉，但能显著影响模型判断。
2. 攻击类型分类
  白盒攻击：攻击者完全了解模型结构、参数和梯度信息
  黑盒攻击：攻击者只能通过API获取模型输出
  有目标攻击：使模型误分类为指定类别
  无目标攻击：只要分类错误即可
3. 常见攻击算法
FGSM (Fast Gradient Sign Method)-快速梯度符号法
原理：沿着梯度上升方向添加扰动，最大化损失函数
公式：x_adv = x + ε * sign(∇x J(x, y_true))
特点：单步攻击，速度快，但扰动较大
PGD (Projected Gradient Descent)-投影梯度下降
原理：FGSM的迭代版本，多步进行
公式：x_adv(t+1) = clip(x_adv(t) + α * sign(∇x J(x_adv(t), y_true)))
特点：攻击能力强，但计算量大
