	  test1:电商用户消费数据分析（缺失值和异常值处理）
 	电商用户消费数据集（含用户ID、消费金额、消费时间、年龄、性别字段），
	其中“年龄”字段存在15%的缺失值，“消费金额”字段存在异常值（远高于正常消费范围）。
	请说明如何处理年龄缺失值和消费金额异常值，写出使用Pandas实现的核心代码。 

	  test2:将分类字段转化为数值特征并编码 
	  某数据集包含"职业"字段（取值为"教师""医生""工程师""程序员"等）和
	"学历"字段（取值为"本科""硕士""博士""大专"），需将这两个分类字段转化为数值特征用于后续建模。
	请说明合适的编码方法，并写出使用Pandas或Scikit-learn实现的代码。

	test3:特征选择、任务分类
	  在一个包含100个特征的分类任务数据集中，部分特征存在高相关性，且可能存在冗余特征。
	  请说明如何进行特征选择（至少两种方法），并写出使用Scikit-learn实现的核心代码。
	
	test4:图像数据转换与降维。
	 现有一份图像数据集（每张图像为28*28像素的灰度图，共10000张），需将其转化为特征矩阵用于机器学习建模，
	且需进行降维处理（保留95%的信息）。请说明数据转换步骤及降维方法，写出使用Numpy和Scikit-learn实现的代码。

	test5:滑动窗口时间特征计算
	 某时间序列数据集（含“日期”和“日销售额”字段），需构建“过去7天平均销售额”“过去3天销售额最大值”
	两个时间特征。请写出使用Pandas实现滑动窗口计算的核心代码。

	test6:客户流失预测模型构建与评估
	 现有一份客户流失预测数据集（特征包括客户时长、月消费、套餐类型等，标签为”是否流失“），
	需构建二分类模型预测客户流失风险。请选择合适的模型（至少两种），写出使用Scikit-learn实现
	“数据划分-模型构建-模型评估”的完整代码，并说明评估指标选择理由。
	
	test7:房价预测随机森林超参数调优
	 在房价预测任务中（标签为“房价”，连续值），使用随机森林回归模型时，需进行超参数调优。
	请说明如何使用“网格搜索”和”随机搜索“两种方法调优，并写出使用Scikit-learn实现的代码。
	
	test8:解决SVM多分类过拟合的方法
	 在多分类任务（如鸢尾花品种分类，3个类别）中，使用SVM模型时，发现模型在测试集上存在过拟合现象。
	请说明至少3种解决过拟合的方法，并写出对应的代码调整方式。
	
	test9:处理信用卡欺诈检测数据的类别不平衡的方法
	 现有一份信用卡欺诈检测数据集（类别极度不平衡，欺诈样本占比1%），使用LightGBM构建分类模型时，
	如何处理类别不平衡问题？请至少写出3种方法及对应实现代码。
	
	test10:模型融合：投票法与堆叠法实现

	请写出使用Scikit-learn 和mlxtend实现的核心代码，并说明两种融合方法的区别。
	
	test11:CNN实现MNIST手写数字分类
	 使用Tensorflow/Keras构建一个简单的卷积神经网络(CNN)，用于MNIST手写数字分类（10个类别，图形尺寸28*28*1）。

	test12:RNN文本分类任务
	 使用Pytorch构建一个循环神经网络（RNN）,用于文本分类任务（如LMDB电影评论情感分析、二分类：正面/负面）。
